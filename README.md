## Курсовая работа по проектированию и разработке 
Сайт груминг-салона

## Стек
React, Vite, Swiper, Sass, JSON-server


## Запуск приложения

```
npm install

npm run dev
```

## Установка и запуск сервера

Установка:

```
npm install -g json-server
```

Запуск :

```
json-server --watch db.json --port 3004
```
